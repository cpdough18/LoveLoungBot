
# Moderation Commands
> Use help <command/category> to see more information about a specific command/category

* `< >` indicates a required parameter
* `( )` indicates an optional parameter

***

## Ban
Bans a user for a given reason

`ban <user> (reason)`

## Kick
Kicks a user for a given reason

`kick <user> (reason)`
## Clear
Clears some messages from the current chat

`clear <count> (reason)`
## Mute
Mutes a user for a given reason

`mute <user> (reason)`
## Unmute
Unmutes a user for a given reason

`unmute <user> (reason)`
## Bulk
Deletes all messages from this channel

`bulk (reason)`
## Reason
Lets you see or edit the reason of a log

`reason <id> <reason>`

`reason <id>`
## Log
Shows the last 10 log items

`log `

`log <category>`

`log <user>`
